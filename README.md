# ArrayList Implementation in Java

A custom implementation of Java's `ArrayList` built completely from scratch using core Java concepts without using Java Collection Framework classes internally.

This project demonstrates how dynamic arrays work internally, including:

- Dynamic resizing
- Adding elements
- Removing elements
- Sorting
- Reversing
- Searching
- Manual memory management logic

---

# 📌 Features

This implementation supports the following operations:

| Operation | Description |
|---|---|
| `add(element)` | Adds an element to the end |
| `add(index, element)` | Adds an element at a specific index |
| `get(index)` | Retrieves element from index |
| `set(index, element)` | Updates element at index |
| `size()` | Returns number of elements |
| `remove(element)` | Removes specific element |
| `remove(index)` | Removes element at index |
| `contains(element)` | Checks if element exists |
| `clear()` | Clears entire list |
| `sort()` | Sorts elements using Bubble Sort |
| `reverse()` | Reverses the list |
